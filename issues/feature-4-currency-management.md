### Title
Implementing currency management 

### Description
This is a multi-currency system and both system level and b2b level currencies should be managed. System admin defines currencies and system parameters and b2b owner specifies default etc.

### Requirements
-[] Currencies should be defined using Admin UI
-[] Each currency have code (iso), symbol, name
-[] Default currency should be set
-[] All currency fields in other tables should keep code of currency
